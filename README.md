# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes retail customer shopping behavior using transactional data to identify purchasing patterns and key factors influencing buying decisions in order to improve sales performance, customer satisfaction, and long-term loyalty.

## Objective
- Analyze customer shopping behavior to identify purchasing patterns and key factors influencing sales.
- Evaluate product and customer performance to support data-driven marketing and business decisions.

## Dataset
Customer Shopping Behavior

## Tools
- Python
- MySQL
- Power BI

## Analysis Approach 
- Clean and prepare data using python
- Analyze customer and product purchasing patterns using MySQL
- Visualize insights and key metrics through an interactive Power BI dashboard

## Conclusions
- Male customers, Young Adults, and Middle-aged groups contribute the largest share of revenue.
- Most customers are Loyal, while New and Returning segments are smaller but still important.
- Non-subscribers generate more revenue and include more repeat buyers than subscribers.
- Certain products rely heavily on discounts, while top-rated items stand out in quality and appeal.
  
## Suggestions
- Offer exclusive perks and tailored benefits to convert repeat buyers into subscribers.
- Reward and recognize repeat buyers to reinforce engagement and expand the Loyal segment.
- Refine discount strategies to maintain sales momentum while protecting profit margins.
- Highlight top-rated and best-selling products, and target high-revenue age groups plus Express shipping users.
